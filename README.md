# Customer Churn Analysis Project

## 📌 Project Overview
This project focuses on analyzing customer churn behavior using Python and Power BI. The main objective of this project is to identify customer patterns, understand churn trends, and create interactive dashboards for business insights.

---

## 🚀 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Power BI
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains customer-related information such as:
- Customer ID
- Gender
- Tenure
- Monthly Charges
- Premium Status
- Customer Support
- Churn Information

---

## 🔍 Project Workflow

### 1️⃣ Data Collection
Loaded customer churn dataset using Pandas.

### 2️⃣ Data Cleaning
- Handled missing values
- Removed duplicates
- Processed categorical columns
- Prepared data for analysis

### 3️⃣ Data Analysis
Performed customer churn analysis to identify:
- High churn customers
- Customer behavior patterns
- Monthly charges trends
- Retention insights

### 4️⃣ Data Visualization
Created an interactive Power BI dashboard including:
- Churn Distribution
- Monthly Charges Analysis
- Customer Tenure Trends
- Premium Customer Insights
- KPI Cards

---

## 📊 Dashboard Insights
- Customers with higher monthly charges showed higher churn risk.
- Customer tenure played an important role in retention.
- Premium customers had different churn behavior patterns.
- Business insights were generated using visual dashboards.
